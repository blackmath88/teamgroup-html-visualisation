# teamgroup-html-visualisation
Query the Teams Intelligence from Graph -> Let an LLM Map it according to your needs into an hmtl-based Visualisation Template = Enjoy the view


## Reusable template

- `visualization_template.html`: generic D3-based template where you only adapt the `TEMPLATE` object (data, modes, dimensions, filters).
- `TEMPLATE_GUIDE.md`: how to adapt the template for governance, collaboration, growth, and cost scenarios.
